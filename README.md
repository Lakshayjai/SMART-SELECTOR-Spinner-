🔽 SmartSelector (Spinner App)
Overview

This app demonstrates the use of Spinner (dropdown menu) to select options from a list.

Features
Dropdown selection
Displays multiple options
Simple UI
Technologies Used
Java
XML
Android Spinner
Project Structure
SpinnerActivity.java → Handles spinner logic
activity_spinner.xml → Layout with spinner
How It Works
Spinner loads predefined options
User selects an option from dropdown
